FILE STRUCTURE DESCRIPTION:  
---------------------------  
  
    THERE ARE A VARIABLE NUMBER OF FIXED-LENGTH RECORDS FOR  
    EACH MILITARY TRAINING ROUTE (MTR).  THE NUMBER OF RECORDS IS  
    DETERMINED BY THE AMOUNT OF TEXTUAL INFORMATION USED TO  
    DESCRIBE THE ROUTE AND THE NUMBER OF POINTS (OR SEGMENTS) ALONG  
    THE ROUTE.  EACH OF THE SIX RECORD TYPES WILL BE IDENTIFIABLE  
    BY A RECORD TYPE INDICATOR (MTR1, MTR2,...,MTR6) AND THE ROUTE  
    TYPE AND IDENTIFIER.  THE RECORDS ARE IN SEQUENCE, I.E., ALL  
    TEXT RECORDS FOLLOW THEIR READABLE ORDER AND ALL SEGMENT  
    RECORDS ARE IN ORDER ALONG THE COURSE OF THE ROUTE.  
    THE  FILE IS SORTED BY ROUTE IDENTIFIER, RECORD TYPE, AND  
    RECORD SEQUENCE NUMBER (INSURES THAT MULTIPLE RECORDS OF  
    A SPECIFIC TYPE ARE IN TEXT/POINT ORDER).  
  
    EACH RECORD ENDS WITH A CARRIAGE RETURN CHARACTER AND LINE FEED 
    CHARACTER (CR/LF). THIS LINE TERMINATOR IS NOT INCLUDED IN THE 
    LOGICAL RECORD LENGTH.

    DESCRIPTION OF THE RECORD TYPES:  
--------------------------------  
  
    'MTR1' RECORD TYPE CONTAINS THE BASE MILITARY TRAINING ROUTE  
    DATA.  THERE IS ONE MTR1 RECORD FOR EACH ROUTE.  
  
    'MTR2' RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING ANY  
    SPECIAL OPERATING PROCEDURES.  EACH MTR2 RECORD  
    CONTAINS ONE LINE OF TEXT.  THERE CAN BE ZERO OR MORE MTR2 
    RECORDS, DEPENDING ON THE LENGTH OF THE PROCEDURE.  
  
    'MTR3' RECORD TYPE CONTAINS TEXTUAL INFORMATION DESCRIBING THE  
    ROUTE WIDTH DIMENSIONS FOR THE ENTIRE ROUTE OR FOR ROUTE  
    SEGMENTS.  EACH MTR3 RECORD CONTAINS ONE LINE OF TEXT.  
    THERE CAN BE 0 OR MORE MTR3 RECORDS.  
  
    'MTR4' RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING  
    TERRAIN FOLLOWING OPERATIONS ALONG THE ROUTE.  EACH  
    MTR4 RECORD CONTAINS ONE LINE OF TEXT.  THERE CAN BE ZERO OR  
    MORE MTR4 RECORDS.  
    
    'MTR5' RECORD TYPE DESCRIBES EACH POINT. IT INCLUDES THE EXACT  
    LATITUDE/LONGITUDE LOCATION OF THE POINT, ITS LOCATION RELATIVE TO  
    A NAVAID, AND TEXTUAL INFORMATION DESCRIBING ALTITUDE, TURNING,  
    AND OTHER SUCH INSTRUCTIONS BOTH LEADING UP TO THE POINT AND  
    AFTER PASSING THE POINT.  

   'MTR6' RECORD TYPE CONTAINS AGENCY INFORMATION. EACH MTR6 RECORD  
    CONTAINS ONE LINE OF TEXT.  THERE CAN BE ZERO OR MORE MTR6 RECORDS. 
  
GENERAL INFORMATION:  
--------------------  
  
     1.  LEFT JUSTIFIED FIELDS HAVE TRAILING BLANKS  
     2.  RIGHT JUSTIFIED FIELDS HAVE LEADING BLANKS  
     3.  ELEMENT NUMBER IS FOR INTERNAL REFERENCE ONLY  
         AND NOT IN RECORD  
     4.  THE MTR ROUTE ID IS MADE UP OF THE ROUTE TYPE AND  
         ROUTE IDENTIFIER. IT IS REPEATED ON EACH RECORD USED  
         TO DESCRIBE THAT ROUTE.
