FILE STRUCTURE DESCRIPTION:
---------------------------
    THERE ARE A VARIABLE NUMBER OF FIXED-LENGTH RECORDS FOR
    EACH AIRWAY(AWY).  THE NUMBER OF RECORDS IS DETERMINED
    BY THE NUMBER OF POINTS AND THE POINT TO POINT ASSOCIATED
    INFORMATION DEFINING THE AWY.  THE AIRWAY FILE CONSISTS
    OF INDIVIDUAL POINTS ALONG AN AIRWAY.  THESE POINTS MAY BE
    NAVIGATION FACILITIES, NAMED FIXES, UNNAMED(NUMBERED)AIRWAY
    CROSSINGS, UNNAMED TURNING POINTS, U.S. BORDERS, OR AIR TRAFFIC
    CONTROL CENTER(ARTCC) BOUNDARY CROSSING POINTS.  DIRECTION
    OF AIRWAY IS WEST TO EAST AND SOUTH TO NORTH.  LAST POINT
    ON AIRWAY IS THE TERMINATION POINT.

    EACH OF THE SIX RECORD TYPES IS IDENTIFIED BY A RECORD
    TYPE INDICATOR (AWY1, AWY2.....AWY5, RMK), AWY DESIGNATION,
    AWY TYPE, POINT SEQUENCE NUMBER (AWY RECORDS ONLY).
    ALL POINTS AND POINT TO POINT DATA IS ARRANGED LOGICALLY
    TO DEFINE THE AIRWAY.  GENERAL REMARK ('RMK') DATA FOLLOWS THE
    DATA FOR THE LAST POINT TO POINT RECORD IN THE ROUTE.
    THE FILE IS SORTED BY AIRWAY TYPE, AIRWAY DESIGNATION,
    POINT SEQUENCE NUMBER, AND RECORD SEQUENCE NUMBER(INSURES
    THAT MULTIPLE RECORDS OF A SPECIFIC TYPE ARE IN
    POINT/REMARKS ORDER).

    EACH RECORD ENDS WITH A CARRIAGE RETURN CHARACTER AND LINE FEED
    CHARACTER (CR/LF). THIS LINE TERMINATOR IS NOT INCLUDED IN THE
    LOGICAL RECORD LENGTH.


DESCRIPTION OF THE RECORD TYPES:
--------------------------------

    'AWY1' RECORD TYPE CONTAINS THE BASIC AND MINIMUM
    ENROUTE ALTITUDE(MEA) DATA FOR EACH AIRWAY POINT/SEGMENT.
    THERE IS ONE AWY1 RECORD FOR EACH AIRWAY POINT/SEGMENT.

    'AWY2' RECORD TYPE CONTAINS THE AIRWAY POINT DESCRIPTION.
    THERE IS ONE AWY2 RECORD FOR EACH AIRWAY POINT.

    'AWY3' RECORD TYPE CONTAINS THE TO CHANGEOVER POINT NAVAID
    DESCRIPTION (VOR AND JET AIRWAYS ONLY).
    THERE IS ONE RECORD FOR EACH TO CHANGEOVER POINT NAVAID DESCRIPTION.

    'AWY4' RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING
    REMARKS ABOUT THE AWY POINT DATA.  EACH AWY4 RECORD
    CONTAINS ONE LINE OF TEXT.  THERE CAN BE 0 OR MORE AWY4
    RECORDS.

    'AWY5' RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING
    EXCEPTIONS TO THE STANDARD AWY CHANGEOVER POINT DATA.  EACH AWY5
    RECORD CONTAINS ONE LINE OF TEXT.  THERE CAN BE 0 OR MORE AWY5
    RECORDS.

    'RMK ' RECORD TYPE CONTAINS TEXTUAL INFORMATION REGARDING
    REMARKS ABOUT AN ENTIRE ROUTE AND NOT A SPECIFIC AIRWAY POINT.
    THERE CAN BE 0 OR MORE RMK RECORDS FOR EACH AIRWAY.
